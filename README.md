# omni-mall


대상

- ChatGPT Apps SDK 기반의 Graph Search 기반 쇼핑몰 MCP 서버 및 위젯 UI 구축
- Olive Young, Daiso, Lotte, Amore Pacific 등 이기종 쇼핑몰을 하나의 공통 스키마로 연결
- 상품 검색, 유사상품 그래프 탐색, 장바구니/체크아웃 연계까지 대화형 AI Shopping 경험 제공

선정 배경

- 쇼핑몰별 API, 상품 속성, 카테고리 체계가 달라 범용 검색/추천 서비스를 빠르게 구축하기 어려움
- 기존 검색은 키워드 일치 중심이라 사용자의 구매 의도와 유사상품 탐색 니즈를 충분히 반영하지 못함
- ChatGPT Enterprise 및 Apps SDK를 활용하면 대화형 검색, 추천, 구매 지원을 하나의 인터페이스로 검증할 수 있어 커머스 업무 확장성과 회사 업무 유관성이 높음

Pain point

- 쇼핑몰별 스키마 차이로 신규 제휴사 온보딩 시 별도 개발과 수작업 매핑이 반복됨
- 검색 결과 정확도와 재현율이 낮아 Zero-result, 낮은 CTR, 낮은 구매 전환이 발생함
- 검색 후 유사상품 탐색 경로가 약해 사용자 이탈이 발생함
- 대화형 검색에서 결제 단계까지의 연결 구조가 부족하고, 상품 메타 품질 편차로 SEO/GEO/AEO 대응이 어려움

Lv3. 과제심사형 결과물 평가 기준		
"단순히 높은(많은) AI요소 기술을 사용 했는가가 아니라, 
해당 문제 해결을 위한 기술 선택이 타당 한지 여부와 그 기술을 활용하여 문제 해결이 실제로 잘 이루어 졌는가를 평가"		
		
세부 평가 기준		
1. 과제 정의		
  ㆍ해결 하고자 하는 명확히 파악하고 적절한 해결방법을 제시하였는가		
  ㆍPain point 해결 여부를 평가 하는데 적절한 성과지표를 설정하였는가		
2. 과제 완성도		
  ㆍ제안서에서 제안한 내용이 충실하게 구현되었는가		
  ㆍ결과물의 완성도가 높은가 		
3. 기술 활용도		
  ㆍ사용 한 기술에 대해 충분히 이해하고 있는가		
  ㆍ목적을 달성하기에 적합한 기술을 사용하였는가		
  ㆍ품질 향상을 위한 최적화가 얼마나 효과적인가		
		
제한 사항		
1. 내 업무는 물론, 회사 사업 영역과도 동떨어진 주제인 경우

4. 단일 오픈소스 프로젝트를 그대로 가져와서 제출하는 경우
